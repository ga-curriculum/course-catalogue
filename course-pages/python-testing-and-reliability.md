| [Selenium: Page Object Model](https://git.generalassemb.ly/modular-curriculum-all-courses/selenium-page-object-model) | Lecture/Lab | 60 min | Introduces the Page Object Model pattern for organizing and maintaining Selenium test code. |
| [Selenium: Data-Driven Testing](https://git.generalassemb.ly/modular-curriculum-all-courses/selenium-data-driven-testing) | Lab | 90 min | Demonstrates using external data sources like CSV or JSON to expand test coverage. |

| [Applying TDD: Red, Green, Refactor](https://git.generalassemb.ly/modular-curriculum-all-courses/tdd-fundamentals-red-yellow-green-lab) | Lab | 120 min | Students practice writing failing tests, implementing minimal solutions, and refactoring safely. |
| [Designing Robust Test Suites](https://git.generalassemb.ly/modular-curriculum-all-courses/designing-robust-test-suites) | Lecture | 95 min | Covers organizing and scaling test suites for maintainability. |
