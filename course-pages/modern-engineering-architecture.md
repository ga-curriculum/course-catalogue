## Content

| Module name | Type | Duration | About |
|---|---|---|---|
| [Intro to the CLI and Bash Scripting](https://git.generalassemb.ly/modular-curriculum-all-courses/intro-to-the-terminal-and-bash-scripting) | Lecture | 45 min | Introduces the terminal as a means to manipulate computer file systems and run programs, explores scripting and running custom script files. |

| [Intro to Cloud Infrastructure](https://git.generalassemb.ly/modular-curriculum-all-courses/intro-to-cloud-infrastructure) | Lecture | 100 min | Provides cloud basics, exploring virtualization, containerization, and the value of IaaS/PaaS/SaaS in modern technology. |
| [The 12 Factor App Methodology](https://git.generalassemb.ly/modular-curriculum-all-courses/the-12-factor-app-methodology) | Lecture | 30 min | Covers cloud-native best practices that guide building scalable, maintainable applications. |
| [Intro to Docker](https://git.generalassemb.ly/modular-curriculum-all-courses/intro-to-docker) | Lecture | 80 min | Explains container fundamentals, Docker architecture, and why containerization streamlines development. |
| [Running your first Docker Container](https://git.generalassemb.ly/modular-curriculum-all-courses/running-your-first-docker-container) | Lecture | 80 min | Guides launching and managing Docker containers for modern engineering workflows. |
| [Building a Node.js Docker Container Lab](https://git.generalassemb.ly/modular-curriculum-all-courses/building-a-nodejs-docker-container-lab) | Lab | 30 min | Hands-on practice containerizing a simple Node.js Express application using Docker. |
| [Container Orchestration](https://git.generalassemb.ly/modular-curriculum-all-courses/container-orchestration) | Lecture | 60 min | Overview of automated deployment and scaling of containerized applications. |
| [Intro to Kubernetes](https://git.generalassemb.ly/modular-curriculum-all-courses/intro-to-kubernetes) | Lecture | 110 min | Introduces Kubernetes core concepts and commands for container management and health checks. |
| [Practical Kubernetes with kubectl and Minikube](https://git.generalassemb.ly/modular-curriculum-all-courses/practical-kubernetes-with-kubectl-and-minikube) | Lecture | 90 min | Demonstrates Kubernetes setup and operation with local clusters, deployments, and debugging. |
| [Infrastructure As Code: Ansible & Terraform](https://git.generalassemb.ly/modular-curriculum-all-courses/infrastructure-as-code-ansible-and-terraform) | Lecture | 120 min | Explores automated infrastructure provisioning using Ansible and Terraform. |
| [Infrastructure As Code: Ansible & Terraform Lab](https://git.generalassemb.ly/modular-curriculum-all-courses/infrastructure-as-code-ansible-and-terraform-lab) | Lab | 45 min | Hands-on creation of an AWS RDS instance with Terraform and optional Ansible configuration. |
| [Intro to Microservices Architecture](https://git.generalassemb.ly/modular-curriculum-all-courses/intro-to-microservices-architecture) | Lecture | 50 min | Introduces microservices principles including loosely coupled services and scalable architectures. |
| [Building a RESTful Microservice with Express](https://git.generalassemb.ly/modular-curriculum-all-courses/building-a-restful-microservice-with-express) | Lecture | 180 min | Walk through building a cloud-connected microservice integrating APIs and infrastructure-as-code elements. |
| [Continuous Integration and Delivery CICD](https://git.generalassemb.ly/modular-curriculum-all-courses/continuous-integration-and-delivery) | Lecture | 90 min | Introduces CI/CD pipelines, testing strategies, and automation for modern software delivery. |
| [Building a CICD Pipeline with Jenkins in Docker](https://git.generalassemb.ly/modular-curriculum-all-courses/building-a-cicd-pipline-with-jenkins-in-docker) | Lecture | 90 min | Guides creating Jenkins and Docker-based pipelines for automated build, test, and deployment. |
| [Test Driven Development with Jest](https://git.generalassemb.ly/modular-curriculum-all-courses/test-driven-development-with-jest) | Lecture | 80 min | Covers TDD principles, unit testing, and Jest basics to improve code quality. |
| [Test Driven Development with Jest and React](https://git.generalassemb.ly/modular-curriculum-all-courses/test-driven-development-with-jest-and-react) | Lecture | 80 min | Extends TDD practices to React applications including component testing and snapshots. |
