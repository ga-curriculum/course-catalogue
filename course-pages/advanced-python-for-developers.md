## Content

| Module name | Type | Duration | About |
|---|---|---|---|
| [Intro to the Terminal and Bash Scripting](https://github.com/ga-curriculum/intro-to-the-terminal-and-bash-scripting) | Lesson | 45 min | Create, copy, move, and remove files from the terminal. |
| [Intro to Git, GitHub, and GitHub Flow](https://github.com/ga-curriculum/intro-to-git-github-and-github-flow) | Lesson | 105 min | Comprehensive introduction to Git and GitHub; covers the GitHub flow process described in GitHub documentation. |
| [Python Requesting Data from APIs](https://github.com/ga-curriculum/python-requesting-data-from-apis) | Lesson | 55 min | Cover API concepts including request URLs, methods, and evaluation while learning to make HTTP requests to external API sources. |
| [Python ExchangeRate API Lab](https://github.com/ga-curriculum/python-exchange-rate-api-lab) | Lab | 60 min | Complete a sample Python application using an Exchange Rate API. |
| [Python API Scavenger Hunt Lab](https://github.com/ga-curriculum/python-api-scavenger-hunt-lab) | Lab | 60 min | Explore public APIs and practice making HTTP requests in Python. |
| [RESTful Routing](https://github.com/ga-curriculum/restful-routing) | Lesson | 50 min | Discuss common API routing patterns. |
| [Intro to Python Virtual Environments](https://github.com/ga-curriculum/intro-to-python-virtual-environments) | Lesson | 30 min | Learn to manage Python virtual environments using Pipenv. |
| [Python Modules & Packages](https://github.com/ga-curriculum/python-modules-and-packages) | Lesson | 30 min | Review Python import syntax and practice importing modules and packages. |
| [MVC Architecture in Python Applications](https://github.com/ga-curriculum/mvc-architecture-in-python-applications) | Lesson | 25 min | Explore the foundational concepts of MVC architecture. |
| [Intro to FastAPI](https://github.com/ga-curriculum/intro-to-fastapi) | Lesson | 45 min | Create a FastAPI application and define your first route. |
| [Python FastAPI MVC CRUD Build](https://github.com/ga-curriculum/python-fastapi-mvc-crud-build) | Lesson | 90 min | Build a full CRUD MVC app with FastAPI using local data. |
| [Intro to PostgreSQL](https://github.com/ga-curriculum/intro-to-postgresql) | Lesson | 65 min | Learn SQL with PostgreSQL. Create databases and tables, manage records, and retrieve data using queries. |
| [Python FastAPI SQLAlchemy Models](https://github.com/ga-curriculum/python-fastapi-sqlalchemy-models) | Lesson | 70 min | Explore SQLAlchemy ORM fundamentals including sessions, models, and seed data. |
| [Python FastAPI SQLAlchemy Relationships](https://github.com/ga-curriculum/python-fastapi-sqlalchemy-relationships) | Lesson | 55 min | Define multiple models, create one-to-many relationships, and update seed data. |
| [Python FastAPI Serializers and Controllers](https://github.com/ga-curriculum/python-fastapi-serializers-and-controllers) | Lesson | 60 min | Define serializers and integrate them with FastAPI controllers. |
| [Python FastAPI Serializers and Controllers Lab](https://github.com/ga-curriculum/python-fastapi-serializers-and-controllers-lab) | Lab | 60 min | Practice defining serializers and controllers for additional models. |
| [Python FastAPI Authentication](https://github.com/ga-curriculum/python-fastapi-authentication) | Lesson | 105 min | Implement authentication with user models, serializers, login routing, and password hashing. |
| [Python FastAPI Authorization](https://github.com/ga-curriculum/python-fastapi-authorization) | Lesson | 110 min | Implement authorization using JWT validation, dependency injection, and protected routes. |
| [Python FastAPI Security SQL Injection](https://github.com/ga-curriculum/python-fastapi-security-sql-injection) | Lesson | 60 min | Demonstrate SQL injection vulnerabilities and implement secure alternatives. |
| [Python FastAPI Unit Testing with PyTest](https://github.com/ga-curriculum/python-fastapi-unit-testing-with-pytest) | Lesson | 50 min | Set up a testing environment and write unit tests for FastAPI routes. |
| [Python FastAPI Unit Testing with Pytest Lab](https://github.com/ga-curriculum/python-fastapi-unit-testing-with-pytest-lab) | Lab | 120 min | Practice writing tests and understanding coverage for common user flows. |
